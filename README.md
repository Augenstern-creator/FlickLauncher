# 🚀 Flick Launcher - 桌面快速启动器

一个轻量级的 Windows 桌面快速启动工具，让你告别在文件夹中翻找的烦恼。

![版本](https://img.shields.io/badge/version-1.0.0-blue)
![平台](https://img.shields.io/badge/platform-Windows-lightgrey)

## ✨ 功能特性

- 🚀 **快速启动** - 一键启动常用软件和文件
- 📁 **分类管理** - 自定义分类，清晰有序
- 🔍 **搜索过滤** - 实时搜索，快速定位
- 🖱️ **拖拽排序** - 自由调整快捷方式顺序
- 🎨 **自定义图标** - 自动提取 .exe 图标，支持自定义
- 📦 **导入/导出** - 一键迁移配置
- 🔄 **开机自启** - 可选开机自动运行
- ⌨️ **全局快捷键** - 按快捷键随时唤出
- 📊 **最近使用** - 记录最近启动的程序
- 🌙 **主题切换** - 亮色/暗色主题

## 🛠️ 安装运行

### 前提条件

- [Node.js](https://nodejs.org/) >= 18
- [pnpm](https://pnpm.io/) >= 8

### 步骤

1. **安装依赖**
   ```bash
   cd flick-launcher
   pnpm install
   ```

2. **开发模式运行**
   ```bash
   pnpm start
   ```

3. **打包为 Windows 安装包**
   ```bash
   pnpm build
   ```
   打包完成后，安装包在 `build/` 目录下。

## 📖 使用指南

### 添加快捷方式

1. 点击右上角 ➕ 按钮
2. 点击「浏览」选择要添加的程序或文件
3. 输入名称（会自动从文件名填充）
4. 选择分类
5. （可选）选择自定义图标
6. 点击「添加」

### 启动程序

- **单击** 快捷方式图标即可启动
- 也可以在「最近使用」区域快速启动

### 管理快捷方式

- **右键** 快捷方式图标 → 打开右键菜单
  - 启动 / 编辑名称 / 更换图标 / 移动分类 / 删除
- **拖拽** 快捷方式图标可以调整顺序

### 分类管理

- 点击分类标签切换显示
- 点击 `+` 创建新分类
- **右键** 分类标签 → 重命名 / 删除

### 设置

- 点击 ⚙️ 打开设置面板
- **主题**: 切换亮色/暗色主题
- **开机自启**: 设置是否开机自动运行
- **全局快捷键**: 选择快捷键组合，按下即可唤出/隐藏启动器
- **导入/导出**: 备份和恢复配置

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+Space` | 显示/隐藏启动器（默认，可修改） |
| `Ctrl+F` | 聚焦搜索框 |
| `ESC` | 关闭弹窗 |

## 📦 数据存储

配置文件存储在：
```
%APPDATA%/flick-launcher/flick-launcher-config.json
```

## 🏗️ 技术栈

- **Electron** - 桌面应用框架
- **HTML/CSS/JS** - 原生前端（无框架依赖）
- **electron-store** - 配置持久化
- **electron-builder** - Windows 打包
